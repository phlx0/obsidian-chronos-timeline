# Screenshot Capture Guide

Take these screenshots in Obsidian with a few dated notes in your vault.
Use a dark theme for the best-looking screenshots.

## Required shots

### 1. `timeline-month.png`
- Open the timeline (ribbon icon)
- Set zoom to **Month**
- Make sure several note cards are visible across different months
- Capture the full Obsidian window (or just the timeline pane)

### 2. `heatmap.png`
- Click **Heatmap** in the toolbar
- Ensure at least one year of notes is visible with varying density
- Capture the heatmap section with the legend visible

### 3. `timeline-swimlanes.png`
- Go to Settings → Chronos Timeline → enable **Group by folder**
- Return to Timeline view
- Capture with multiple swimlane bands visible

### 4. `demo.gif`
An animated GIF showing:
1. Opening the timeline from the ribbon
2. Scrolling left/right
3. Zooming in (Month → Week)
4. Switching to Heatmap
5. Dragging a card to reschedule

Recommended tool: LICEcap (Windows/Mac) or Kap (Mac)
Recommended dimensions: 1200×700px, 15fps, 20-30 seconds

## Tips
- Use Obsidian's built-in screenshot: Ctrl+Shift+S (may vary by OS)
- Rename files exactly as listed above
- Place them in this folder (docs/screenshots/)
- Then update README.md image paths if needed
